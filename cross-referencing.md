# Cross-Referencing

Eine der herausragenden Funktionen von Docupulse ist die **Cross-Referencing Funktion**. Sie können eine bestimmte Zelle oder einen Bereich in Excel (z.B. ein Kostenstelle oder eine Dividendenberechnung) mit der genauen Stelle in einem PDF- oder Word-Dokument verknüpfen, das diese Daten unterstützt.

## Beispiele

- **Kostenanalyse**: Verweisen Sie auf eine Dienstleistungsvereinbarung eines Lieferanten, sodass Sie immer die genaue Quelle jedes Kosteneintrags kennen.
- **Gesellschaftervereinbarung**: Verbinden Sie eine Klausel, die die Bedingungen der Dividendenausschüttung beschreibt, mit der entsprechenden Excel-Zelle, die die Auszahlungen berechnet.

## Vorteile

- **Dynamische Verknüpfung**: Verknüpfen Sie Excel-Zellen, Pivot-Tabellen oder Bereiche direkt mit bestimmten Abschnitten eines beliebigen PDF- oder Word-Dokuments.
- **Erhöhte Transparenz**: Wenn Sie auf eine Zahl (z.B. Kosten, Umsatz oder Klauseln in einem Vertrag) verweisen, können Sie direkt zum genauen Standort im Quell-Dokument springen.
- **Integrierter Viewer**: Sehen Sie vollständige Dokumente direkt in Excel, springen Sie zu relevanten Auszügen und korrelieren Sie diese mit Ihren Daten.
